在Ubuntu 16.04 中安装配置Tomcat

发表于 2018-03-14 | 分类于 技术 |

简介

Tomcat 服务器是一个免费的开放源代码的Web 应用服务器，属于轻量级应用服务器，在中小型系统和并发访问用户不是很多的场合下被普遍使用。
MySQL 是一种关系数据库管理系统，关系数据库将数据保存在不同的表中，而不是将所有数据放在一个大仓库内，这样就增加了速度并提高了灵活性。
所以本文选用Tomcat作为web服务器，MySQL作为数据库系统来运行web项目。

安装环境

  • 平台: Ubuntu 16.04
  • 软件: jdk1.8.0_161 | tomcat-9.0.6

配置Java环境

  1. 在 /usr/local 下新建文件夹Java
    cd /usr/local
    sudo mkdir Java

  2. 下载 jdk 并将下载好的 jdk 文件拷贝至 /usr/local/Java 文件夹


  3. cd 至 jdk 所在文件夹中
    cd /usr/local/Java

  4. 解压 jdk
    sudo tar -xzvf jdk-8u161-linux-x64.tar.gz
  5. 删除 jdk 压缩包
    sudo rm jdk-8u161-linux-x64.tar.gz

  6. 开始配置 Java 环境变量

    • 配置 /etc/profile 文件
      sudo vim /etc/profile（如果未安装vim，请用以下命令安装vim sudo apt-get update | sudo apt-get install vim）

    • 在末尾添加以下内容(根据自己Jdk安装路径及版本自行调整)

    • 使修改生效
      source /etc/profile

  7. 至此 Java 环境已经配置完成，可以通过以下命令来检查
    java -version
    或
    javac

配置Tomcat服务器

  1. 在 /usr/local 下新建文件夹 Tomcat
    cd /usr/local
    sudo mkdir Tomcat

  2. 下载 Tomcat 并将下载好的 Tomcat 文件拷贝至 /usr/local/Tomcat 文件夹


  3. cd 至 Tomcat 所在文件夹中
    cd /usr/local/Tomcat

  4. 解压 Tomcat
    sudo tar -xvf apache-tomcat-9.0.6.tar
  5. 删除 Tomcat 安装包
    sudo rm apache-tomcat-9.0.6.tar

  6. 启动 Tomcat 服务

    • cd 至 bin 文件夹
      cd apache-tomcat-9.0.6/bin/

      如果提示没有权限，请用以下命令以root用户登录继续访问
      sudo su
      以root用户登录后可能要重新使java变量生效
      source /etc/profile

    • 启动服务
      ./startup.sh

  7. 至此 Tomcat 服务器已经配置完毕，打开浏览器通过 localhost:8080 访问服务器，出现以下图片表示配置成功

配置Mysql

  1. 通过ubuntu的软件包管理器安装Mysql
    sudo apt-get update
    sudo apt-get install mysql-server

  2. 期间会为mysql服务配置密码（可自行设置）

phpMyAdmin（可选）

简介

phpMyAdmin 是一个以PHP为基础，以Web-Base方式架构在网站主机上的MySQL的数据库管理工具，让管理者可用Web接口管理MySQL数据库。借由此Web接口可以成为一个简易方式输入繁杂SQL语法的较佳途径，尤其要处理大量资料的汇入及汇出更为方便。

配置

  1. 首先配置apache服务器
    sudo apt-get install apache2
  2. 安装phpMyAdmin，安装时选择 apache服务器（期间会配置用户名密码）
    sudo apt-get install phpmyadmin

  3. 全部安装好以后，建立软连接
    sudo ln -s /usr/share/phpmyadmin /var/www/html/phpmyadmin
  4. 重启apache服务器
    sudo /etc/init.d/apache2 restart
  5. 至此phpMyAdmin就算配置完成，通过localhost/phpmyadmin访问即可
