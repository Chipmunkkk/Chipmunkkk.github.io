Java中的数据类型

发表于 2018-04-26 | 分类于 读书笔记 |

概述

Java作为一种强类型语言，必须为每一个变量声明一种类型。Java种的数据类型分为两大类：基本数据类型、引用数据类型。

基本数据类型可以分为三类数据类型：数值类型、字符类型以及布尔类型。同时又可以分为八种数据类型：4种整型、2种浮点类型、1种字符类型以及1种布尔类型。

引用数据类型有：数组、类以及接口。

Java种数据类型的取值范围是固定的，不会随着机器硬件环境的不同而产生差异。

基本数据类型

变量名指向具体的数值的数据类型我们称之为基本数据类型。在声明基本数据类型之后，Java会立即为其分配内存空间。

数值类型

整型

整型就是指我们所理解的数学概念上的整数，可以为负数。Java一共为我们提供了4种整型（int，short，long，byte）

int

int 是编程种最常用的一种整型，每存储一个int类型的数据，需要占用4个字节。

  • 最大值为：2,147,483,647（2^31 - 1）
  • 最小值为：-2,147,483,648（-2^31）
  • 默认值为：0

short

short 是一种短整型，一个short类型的变量，需要占用2个字节。

  • 最大值为：32,767（2^15 - 1）
  • 最小值为：-32,768（-2^15）
  • 默认值为：0

long

long 是一长整型，通常用来存储比较大的整数。一个long类型的变量，需要占用8个字节。

  • 最大值为：9,223,372,036,854,775,807（2^63 - 1）
  • 最小值为：-9,223,372,036,854,775,808（-2^63）
  • 默认值为：0L （通常在值后面加一个大写的英文字母L）

byte

byte是Java中最小的数据类型。一个byte类型的变量，仅占用1个字节。

  • 最大值为：127（-2^7 - 1）
  • 最小值为：-128（-2^7）
  • 默认值为：0

浮点类型

浮点类型用于表示有小数部分的数值，也就是我们通常说的小数。Java一共为我们提供了2种浮点类型（float，double）

float

float为单精度浮点数，运行速度相比double更快，占内存更小，但是当数值非常大或者非常小的时候会变得不精确。一个float类型的变量，存储需要4个字节。

  • 取值范围：大约± 3.402 823 47E+38F（有效位数6～7位）
  • 默认值为：0.0f

double

double为双精度浮点数。一个double类型的变量，存储需要8个字节。

  • 取值范围：大约± 1.797 693 134 862 315 70E+308（有效位数15位）
  • 默认值为：0.0d（d 可省略）

字符类型

char

char 类型是一个单一的 16 位 Unicode 字符，用来表示单一字符。

  • 最大值为：\uffff（即为65,535）
  • 最小值为：\u0000（即为0）
  • char 类型可以储存任何字符；

布尔类型

boolean

boolean类型只有两个值：true、false，用来判定逻辑条件。

  • 默认值为：false

基本数据类型之间的转换

自动类型转换

数值类型与字符类型可以混合运算。在运算中，不同类型的数据先转化为同一类型，然后进行运算。

低 ————————————————> 高
byte，short，char —> int —> long—> float —> double

数据类型转换必须满足如下规则：

  • 不能对boolean类型进行类型转换。
  • 不能把对象类型转换成不相关类的对象。
  • 在把容量大的类型转换为容量小的类型时必须使用强制类型转换。
  • 转换过程中可能导致溢出或损失精度。
  • 浮点数到整数的转换是通过舍弃小数得到，而不是四舍五入。

强制类型转换

当然，我们并不是一直想要int类型转换为double类型，可能某一天我们想把double转换成int类型。Java也想到了这一点，所以它支持蕾丝这种转换，我们称之为强制类型转换。

  • 条件是转换的数据类型必须是兼容的。
  • 格式：(type)value type是要强制类型转换后的数据类型。

如果在强制类型转换的过程中，超出目标类型的范围，则结果会被截断成为一个完全不同的值，比如 (byte)300 结果为44。

引用数据类型

变量名指向存数据对象的内存地址的数据类型我们称之为引用数据类型。它以特殊的方式（类似C指针）指向对象实体（具体的值），这类变量声明时不会分配内存，只是存储了一个内存地址。

引用类型一般是通过new关键字来创建，它存放在内存的堆中，可以在运行时动态的分配内存大小，生存期也不必事先告诉编译器，当引用类型变量不被使用时，Java内部的垃圾回收器GC会自动回收走。

String类型

其他常见引用数据类型

每一种基本数据类型在Java中都有对应的封装类型，也就是引用类型：

  • 整数类型：Integer、Short、Long、Byte
  • 浮点数类型：Float、Double
  • 字符型：Character
  • 布尔类型：Boolean
